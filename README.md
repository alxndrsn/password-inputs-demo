password-inputs-demo
====================

Demo React app showing a complicated password change page.

# Features

* checks password length
* checks new password is different to old password
* confirms 2 new password fields match
* estimates password strength with [`zxcvbn`](https://www.npmjs.com/package/zxcvbn)
* checks [haveibeenpwned](https://haveibeenpwned.com/) password database for inclusion in breaches
